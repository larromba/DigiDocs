==== Name ====

DigiDocs

==== Description ====

"DigiDocs" является самым простым из возможных способов сканирования документов и отправить их на свой компьютер через электронную почту или десантный. Если вы хотите «оцифровать» и избавиться от бумажных документов, это приложение для вас.

Особенности:
- Очень быстро захватить несколько страниц и сохранить их в PDF-документе.
- Отправка документов на свой компьютер
- Документы сохраняются в памяти телефона для последующего использования
- Файл менеджеры включен, чтобы освободить память
- Полезно для квитанций

Совместимость с IPad и iPhone. Оптимизирован для iPhone 6.
